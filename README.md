# Clínica Borcelli

Site institucional moderno desenvolvido para a **Clínica Borcelli **, com foco em apresentação profissional, experiência do usuário e design responsivo.

O projeto foi construído utilizando **HTML e CSS**, com uma interface elegante que valoriza os serviços e a identidade visual da clínica.


## Visão Geral

Este site apresenta informações completas da clínica, incluindo:

* Página inicial com destaque visual (Hero)
* Seção "Sobre Nós"
* Cards de procedimentos
* Lista com imagem ilustrativa
* Vídeo institucional
* Localização da clínica
* Rodapé com informações de contato



## Funcionalidades

* Layout moderno e profissional
* Organização clara das informações
* Uso de imagens e elementos visuais
* Integração com vídeo institucional
* Seção de localização da clínica
* Navegação simples e intuitiva

## Tecnologias Utilizadas

* HTML5
* CSS3
